## 📌 About the Project

The **SOC Log Analyzer** is a cybersecurity-focused web application designed to help Security Operations Center (SOC) analysts quickly analyze and detect suspicious activities from log data.

This tool allows users to upload log files and automatically identifies potential threats, anomalies, and patterns that could indicate security incidents.

---

## 🎯 Key Objectives

* Simplify log analysis for beginners and L1 SOC analysts
* Detect suspicious patterns in logs
* Provide quick insights into potential security threats

---

## ⚙️ Features

* 🔍 Upload and analyze log files (CSV format)
* 🚨 Detect suspicious IP addresses and activities
* 📊 Display insights in an easy-to-read format
* ⚡ Fast and user-friendly interface
* 📈 Helps in basic threat detection and monitoring

## ▶️ How to Run Locally

### 1. Clone the Repository

```
git clone https://github.com/Santhujuloori/SOC-Log-Analyzer.git
cd SOC-Log-Analyzer
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Run the Application

```
streamlit run app.py
```
